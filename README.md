# JavaScript Upskilling Exercises

This repository contains JavaScript exercises completed as part of the Cognizant Digital Nurture 5.0 Upskilling Program.

## Topics Covered

* JavaScript Basics and Setup
* Syntax, Data Types, and Operators
* Conditionals, Loops, and Error Handling
* Functions, Scope, Closures, and Higher-Order Functions
* Objects and Prototypes
* Arrays and Array Methods
* DOM Manipulation
* Event Handling
* Async JavaScript, Promises, and Async/Await
* Modern JavaScript (ES6+)
* Working with Forms
* AJAX and Fetch API
* Debugging and Testing
* jQuery Fundamentals

## Project Theme

**Local Community Event Portal**

The exercises are based on a community event portal where users can view events, register for activities, filter events, and interact with dynamic web pages using JavaScript.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* jQuery

## Learning Outcomes

Through these exercises, I gained hands-on experience in:

* Writing and organizing JavaScript code
* Manipulating the DOM dynamically
* Handling user interactions and events
* Working with objects, arrays, and functions
* Performing asynchronous operations using Fetch API
* Validating forms and handling errors
* Debugging and testing web applications
